# Usage

On Home dir:
    $ git clone git://github.com/lsdr/vim-folder.git .vim
    $ ln -s .vim/_vimrc .vimrc
    $ ln -s .vim/_gvimrc .gvimrc

To update:
    $ cd .vim
    $ git pull origin master
